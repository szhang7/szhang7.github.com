## TECHNICAL TERMS

###AJAX
Asynchronous JavaScript and XML (异步JavaScript和XML)

###AOP
Aspect-oriented Programming (面向切面编程)

###CDN
Content Delivery Network (内容分发网络)

###CGI
Common Gateway Interface (公共网关接口)

###CRUD
Create, Retrieve, Update, Delete (增加、查询、更新、删除)

###DAO
Data Access Object (数据访问对象)

###DDD
Domain-Driven Design (领域驱动设计)

###DOM
Document Object Model (文件对象模型)

###HQL
Hibernate Query Language (Hibernate查询语言)

###HTML
Hyper Text Markup Language (超文本标记语言)

###HTTP
Hyper Text Transfer Protocol (超文本传输协议)

###IoC
Inversion of Control (控制反转)

###JAXB
Java Architecture for XML Binding(用于XML绑定的Java体系结构)

###JDBC
Java Data Base Connectivity (Java数据库连接)

###JPA
Java Persistence Annotations (Java持久性注解)

###JSON
JavaScript Object Notation (JavaScript对象符号)

###JSP
Java Server Page (Java服务器页面)

###MVC
Model View Controller (模型、视图、控制)

###NPM
NodeJS Pakcage Management (NodeJS包管理)

###POJO
Plain Ordinary Java Objects (简单的Java对象)

###POM
Project Object Model (项目对象模型)

###REST
Representational State Transfer (表达性状态转移)

###SCM
Source Code Management (源代码管理)

###W3C
World Wide Web Consortium (万维网联盟)

###WTP
Web Tools Platform (Web工具平台)

###WWW
World Wide Web (环球信息网，简称Web，中文名字为万维网)

###XML
eXtensible Markup Language (可扩展标记语言)
